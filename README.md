# E-commerce Recruitment Task

A modern, responsive product listing page built with **Next.js 15** and **TypeScript**. This project focuses on performance, clean architecture, and optimal user experience.

🔗 **Live Demo:** [https://recruitment-task-hg43.vercel.app/](https://recruitment-task-hg43.vercel.app/)

---

## 🛠 Tech Stack & Decisions

- **Next.js 15 (App Router):** Leveraged for its efficient rendering and built-in optimizations.
- **SCSS Modules:** Used for styling to ensure scoped, maintainable, and robust CSS without global namespace conflicts.
- **Context API:** Implemented for shopping cart management to ensure a smooth data flow and effectively avoid **prop drilling**.
